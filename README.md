# Morena Children's Village
## Introduction
Morena Children's Village is a charity website designed to provide information and resources for helping children. The website is built using HTML5, CSS3, and JavaScript, and features a responsive design for optimal viewing on various devices.
## Key Features
- Responsive design for mobile, tablet, and desktop devices
- Charity-focused content, including information on donating and volunteering
- Gallery section for showcasing images and videos
- Contact page for getting in touch with the organization
## Tech Stack
- HTML5
- CSS3
- JavaScript
- Bootstrap
- Animate.css
- Icomoon Icon Fonts
## Installation
1. Clone the repository to your local machine
2. Open the index.html file in a web browser to view the website
## Usage
- Navigate through the website using the menu links
- View the gallery section to see images and videos
- Fill out the contact form to get in touch with the organization
